    Responsive Website

    This is very important, especially in web design, so that the display that has been created looks structured and not messy when viewed in terms of size, such as desktop, mobile and tablet.

    Did you know that data from Statista in the second quarter of 2022 shows that mobile device users dominate 59% of all website traffic sources in the world. Considering this, responsive web design (RWB) should be applied to all types of websites.

    The first requirement for a responsive display can be seen from the layout. Basically, a responsive web must have a website layout design that is neatly arranged even though it is accessed from various different devices. If the layout still looks messy, then the site cannot be called a responsive website.
